## What Droid Transfer Does

Droid Transfer is designed to connect Android devices with desktop workflows, giving you a single place to move data in both directions without relying on cloud services in the middle. It can be used for backing up important conversations, keeping media collections in sync, or aligning contact records, all through an interface that remains usable across different screen sizes and operating systems.

Version 1.80 adds protocol-native encryption for every transfer, helping keep personal data protected during sync sessions. The release also includes multilingual localization and a built-in 24/7 help system, making the app easier to use for people in different regions and reducing the amount of setup knowledge required.

## Usage

Connect your Android device with a USB cable, or make sure both devices are on the same Wi-Fi network. Start Droid Transfer and choose the data categories you want to synchronize:

```
# Example workflow for contact backup
1. Launch Droid Transfer
2. Select "Contacts" from the main menu
3. Choose "Export to CSV" or "Sync to Desktop"
4. Confirm destination folder
5. Begin transfer
```

When transferring media, the transcoding pipeline automatically adjusts unsupported formats so the content can play on the destination device.

## FAQ

**Q: How do I update from an older version?**  
A: Download the latest release and install over your existing installation. Your configuration and data will be preserved.

**Q: Can I transfer data without USB?**  
A: Yes, Droid Transfer supports local network transfers when both devices are connected to the same Wi-Fi network.

**Q: What encryption standard is used?**  
A: Protocol-native encryption employs AES-256 for data in transit. Details are available in the documentation.

**Q: The application does not detect my device.**  
A: Ensure USB debugging is enabled on your Android device and that appropriate drivers are installed. For network transfers, verify both devices are on the same subnet.
